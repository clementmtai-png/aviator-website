# Pesapal API reference

Quick reference for Pesapal API 3.0 integration.

---

## Postman collection

**URL:** [https://documenter.getpostman.com/view/6715320/UyxepTv1](https://documenter.getpostman.com/view/6715320/UyxepTv1)

Use this collection to explore and test Pesapal endpoints (Authentication, SubmitOrderRequest, GetTransactionStatus, etc.).

---

## Base URLs

| Environment | URL |
|-------------|-----|
| **Sandbox** | `https://cybqa.pesapal.com/pesapalv3` |
| **Live**    | `https://pay.pesapal.com/v3` |

Set `PESAPAL_SANDBOX=true` in your env to use Sandbox; omit or set to `false` for Live.
